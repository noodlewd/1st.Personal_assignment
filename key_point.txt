1. 검색 기능 구현(★★★)
    - 글자를 입력하거나, enter를 동작할 시 알맞는 키워드의 영화리스트가 나열되어야 함
    - 리스트가 나열되는 속도 조절 해야 함

2. grid layout(★★★)
    - 그리드 레이아웃 안에 가져온 api정보가 들어가야 함
    - 클릭 시 modal창이 뜨도록 구현해야 함

3. modal창(★★★★)
    - modal창 안에 상세 내용이 들어가야 하며, 북마크 추가 기능 및 모달창 닫는 기능 들어가야 함

4. 북마크(★★★★★)
    - modal을 통하여 북마크 표시한 영화들 받아와야 함
    - 북마크 내부의 modal창은 북마크 삭제 기능 구현해야 함
